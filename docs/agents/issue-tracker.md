## Agent workflow

Agents cannot push cards to Trello automatically. Instead:

1. The agent outputs a formatted card description (title, description, labels, checklist).
2. You copy it into a new Trello card manually.
3. Apply the label from `triage-labels.md` that matches the card's current state.
